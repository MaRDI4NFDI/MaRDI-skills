# MaRDI-skills

> 🚧 **Under construction** 🚧
>
> This marketplace is a work in progress and should not be used yet.

A Claude Code skill marketplace for the [Mathematical Research Data Initiative (MaRDI)](https://www.mardi4nfdi.de/).

This repository bundles reusable [skills](https://docs.claude.com/en/docs/claude-code/skills) — small, focused capabilities that AI coding agents can load on demand for tasks that recur across MaRDI projects (data curation, mathematical research workflows, package development, documentation, etc.).

## Installation

### Claude Code

Run `/plugin` in Claude Code, add this repository as a marketplace, and install the plugins you want. Plugins are enabled globally for your user.

### Other agents (Codex, Cursor, Cline, …)

Use the [`npx skills`](https://github.com/vercel-labs/skills) command line tool for managing and installing skills.
